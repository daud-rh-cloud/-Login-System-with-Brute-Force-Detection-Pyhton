# -Login-System-with-Brute-Force-Detection-Pyhton

A simple Python terminal-based login system built as a learning project.

📋 Features

Register a new user with a username and password
Password Strength Check — minimum 8 characters and at least one uppercase letter
Login with username and password
Brute Force Protection — account gets locked after 6 failed login attempts
View all users and their passwords
Exit the program cleanly


🚀 How to Run
Make sure you have Python installed, then run:
python main.py

🗂️ Menu Options
=== Login System ===
1. Register
2. Login
3. Show all users
4. Exit

🔑 Password Rules
When registering, your password must:

Be at least 8 characters long
Contain at least one uppercase letter

If not, you will be asked to enter a new password.

🔒 Brute Force Protection

Every failed login attempt is tracked per user
After 6 failed attempts, the account is permanently locked
Locked accounts cannot log in even with the correct password
